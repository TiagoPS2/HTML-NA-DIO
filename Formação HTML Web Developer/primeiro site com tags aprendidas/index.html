Tag <font>

A tag <font> é suportada até o HTML4 com o intuito de estilizar os textos nos sites como: color, face, size. Porém de forma obsoleta na qual possuimos atualmente.

Example 1:
font color is red using a named color

Example 2:
font face is a different font family

Example 3:
font size is 5 using numeric value

Tag <del>

Um texto entre as tag <del> define o texto que foi excluído de um documento. Os navegadores geralmente traçam uma linha através do texto excluído e se usar a tag <ins> o texto entre essa tag será colocado no lugar.

Example:
My favorite color is blue red!

Tag <abbr>

A <abbr> tag define uma abreviação ou um acrônimo, como "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".

Example 1:
A WHO foi fundada em 1948.

Example 2:
O HTML surgiu em 1991.
